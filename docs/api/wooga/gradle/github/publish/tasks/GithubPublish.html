              GithubPublishSpec

              class GithubPublish
extends AbstractGithubTask

              Publish a Github release with or without provided assets.

              The task implements org.gradle.api.file.CopySourceSpec and org.gradle.api.tasks.util.PatternFilterable. Assets to restore can be specified via copy spec syntax.

              Example: 

               githubPublish {
         targetCommitish = "master"
         tagName = project.version
         releaseName = project.version
         body = "Release XYZ"
         prerelease = false
         draft = false
         publishMethod = "create"
         from(file('build/output'))
     
}
